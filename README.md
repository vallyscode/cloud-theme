## Installation

#### Melpa

#### Manual

```
    M-x package-install RET cloud-theme
    (load-theme 'cloud-theme t)
```

#### Via use-package

```
    (use-package cloud-theme
        :ensure t
        :config
        (load-theme 'cloud t))
```
